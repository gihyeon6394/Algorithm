## 2장 정렬

### 정렬이란?

- 나열된 순서를 논리적인 순서에 맞게 바꾸는 작업
- e.g. 신용카드 거래내역 (날짜 기준 정렬)

### 정렬의 중요성

- 알고리즘 비교방법의 시작
- 다른 문제들에도 정렬 테크닉 응용 가능
- 문제 풀이시 정렬부터 시작해야하는 경우가 많음

### 정렬의 상업 분야

- 데이터 처리
- 대량 트랜잭션 처리
- 조합 최적화
- 유전자 분석
- 날씨 에측 등
