## 3장 탐색 알고리즘

- 현대 컴퓨팅 기술과 인터넷은 어마어마한 양의 정보에 접근하게 해줌
- 많은 정보를 효율적으로 탐색하는 것은 정보처리에 있어 꼭 필요한 기초 작업
- 심볼 테이블 (Symbol table) : 어떤 정보 (값)를 저장하고 다시 key를 통해 정보에 접근하는 추상적인 메커니즘
    - 사전, 색인이라고도 함
    - 책의 색인과 기능이 비슷 (마치 국어사전의 단어가 키인 것처럼)
